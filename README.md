#  E-commerce Sales Dashboard (Power BI)

##  Overview

This project presents an interactive Power BI dashboard built to analyze e-commerce sales performance and customer behavior.

The dashboard provides key business insights through revenue tracking and customer segmentation using RFM (Recency, Frequency, Monetary) analysis.

---

## Key Features

* KPI Cards showing **Total Revenue** and **Total Customers**
* Monthly Revenue Trend Analysis
* Customer Segmentation using RFM (High, Mid, Low Value)
* Interactive and clean dashboard design

---

##  Insights

* Majority of customers fall into the **mid-value segment**
* Opportunity to convert mid-value customers into high-value customers
* Clear visibility of monthly revenue performance

---

##  Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Modeling

---

## Files Included

* `ecommerce-dashboard.pbix` → Power BI dashboard file
* `ecommerce_data.csv` → Raw dataset
* `monthly_revenue.csv` → Aggregated monthly data
* `rfm_analysis.csv` → Customer segmentation data
